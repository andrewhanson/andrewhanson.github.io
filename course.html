Bootstrapping ASP.NET Core

Enough tutorials already.. build something!

.NET Core and ASP.NET Core give you a blazing fast and modular platform for creating server applications that run on Windows, Linux and Mac - Microsoft Marketting Site

Do you really have time to investigate everything there is to know about developing ASP.Net Core, cross platform, .net standard, Xamarin applications running on a Mac using a docker image deployed to Microsoft Azure with Visual Studio Code leveraging the latest DotNetCompilerPlatform with a self-contained developer mode and the ability to deploy to a cheap Linux VM host.
…. And that’s before you even get started with Javascript or actually delivering anything of value to your customers.

Seriously are you a robot!

If you are anything like me you might actually want to spend some time away from your computer once you’re done working. But at the same time you don’t consider the Visual Studio -> File New dialog to actually be a realistic start to a new project.

Take this course if you are:

  • Interested in ASP.NET Core but actually have a real job
  • New to the .Net developer platform and just want to kick the tires on the latest model
  • You care more about delivering value to your users then clever programming tricks
  • Willing to share your code publicly

Do not take this course if:
  • You can enumerate all the different version of the .Net Framework with using Google Bing
  • You don’t actually want to build working software
  • You like to compile all your dependent libraries from source.
  • Want to keep your work to yourself

Not Another Programming Course

All the information you need to build ASP.Net Core applications is already out there. But I love the quote from Derek Sivers where he says,
“If information were the answer, we’d all be billionaires with perfect abs”

Or in our case, “If information were the answer, we’d all be expert ASP.Net Core developers” ready and able to wow our customers with excellent software delivered onetime and under budget.

You need to know the fastest way to get started and where to go for more information when you project takes you off the beaten path.

You Time is your most valuable Asset

You can always make more money and more friends but you can never make more time. You need a course that doesn’t waste your time defining what CSS is, how to declare a variable, use GitHub or opine the intricate beauty of Dependency Injection. You need to have a technology explained once clearly and then forced to develop some working product using that technology.

Are You Committed Enough to Put In the Work

It doesn’t matter how concise and clear the information and samples in the Bootstrapping ASP.Net Core course are created if you are not willing to put in the work to actually create on your own. To benefit from this course you have to be able to put in the work. Do you actually have enough discipline to work?

In the Bootstrapping ASP.NET Core course you will be given information and the accountability necessary to actually put that information into practice. That’s right just like high school there will be homework and it will be checked.
